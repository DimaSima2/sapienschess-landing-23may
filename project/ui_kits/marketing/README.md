# SapiensChess — Marketing UI Kit

A hypothetical partner-program landing page distilled from the pitch deck. No website exists in the source material — this kit extends the deck's visual language to the web so the brand can be used beyond slides.

## Components

| File | What it is |
|---|---|
| `index.html` | Full one-page landing — Header, Hero, Audience strip, Format, Benefits, Contact, Footer. |
| `Header.jsx` | Top bar — wordmark left, nav center, "Стать партнёром" pink CTA right. |
| `Hero.jsx` | Big pink ALL-CAPS headline, neutral lede, primary CTA, B&W tournament photo. |
| `AudienceStrip.jsx` | Pink section header + 4-up audience chips. |
| `FormatBlock.jsx` | "МЫ / ВЫ" partnership split with bullets. |
| `BenefitsGrid.jsx` | Two-column benefits panel + photo rail. |
| `Contact.jsx` | Director card + phone + QR placeholder. |
| `Footer.jsx` | Wordmark + stripe + © line. |
| `Button.jsx`, `Stripe.jsx` | Shared primitives. |

## Design rules respected

- **Hard edges only** — `border-radius: 0`. No shadows.
- **Hot-pink** for all headlines + primary CTAs.
- **B&W photography** with desaturation filter.
- **Roboto** weight contrast (900 / 700 / 400) carries the type hierarchy.
- **Stripe motif** at top of page and footer.
- **No emoji, no gradients, no glass.**

## Notes / caveats

This is a recreation from the deck's visual DNA — no production source code exists. Components are cosmetic recreations, not real implementations.
